# yt-dlp

## MacOS

開啟終端機，安裝 [Homebrew](https://brew.sh/)

輸入以下指令安裝 yt-dlp 和 ffmpeg：

```
brew install yt-dlp ffmpeg
```

日後更新指令：

```
brew upgrade yt-dlp
```

確認軟體版本：

```
yt-dlp --version
ffmpeg -version
```
